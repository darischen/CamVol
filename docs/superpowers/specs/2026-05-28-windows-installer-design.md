# HandVol Windows Installer Design

**Date:** 2026-05-28  
**Status:** Approved  
**Goal:** Create a one-time game-like installer (.exe) that bundles Python, MediaPipe model, and HandVol code, auto-configures Windows Task Scheduler, and requires no user setup.

---

## Overview

Users download `HandVol-Installer.exe` (~300-400MB), run it, answer a few wizard prompts, and HandVol is installed and ready to auto-run at next boot. No command line, no manual configuration, no prerequisites beyond Windows 10/11 with admin privileges.

---

## Architecture

### Components

1. **NSIS Installer Script** (`handvol-installer.nsi`)
   - Wizard UI with install path, optional auto-launch, optional desktop shortcut
   - Extracts bundled Python, MediaPipe model, HandVol code to `C:\Program Files\HandVol\`
   - Calls `install_task.bat` to register Windows Task Scheduler entry
   - Handles uninstaller generation

2. **Bundled Runtime & Assets** (in `build/` during build process)
   - Embeddable Python 3.11+ runtime (self-contained, no system Python needed)
   - MediaPipe gesture recognizer model (tracked in Git LFS, ~200MB)
   - HandVol source code with all pip dependencies pre-installed

3. **Task Scheduler Setup Script** (`install_task.bat`)
   - Runs during install to create Windows Task Scheduler entry
   - Task name: `HandVol`
   - Trigger: System startup
   - Action: Run `pythonw.exe C:\Program Files\HandVol\handvol.pyw`
   - Runs with highest privileges (for audio control)
   - Runs hidden (no console window)

4. **Uninstaller** (auto-generated by NSIS)
   - Accessible via Add/Remove Programs
   - Removes Task Scheduler entry
   - Deletes `C:\Program Files\HandVol\`
   - Removes desktop shortcut if created
   - Cleans registry entries

## Installer Workflow

### Before Build
1. Track MediaPipe model in Git LFS (`models/gesture_recognizer.task`)
2. Create NSIS script and Task Scheduler batch script
3. Document build process for reproducibility

### Build Process
1. Download/cache embeddable Python 3.11 (if not cached)
2. Copy MediaPipe model from repo (via Git LFS)
3. Copy HandVol source code
4. Install all pip dependencies into bundled Python runtime
5. Compile NSIS script → generates `HandVol-Installer.exe`

### User Wizard Flow
1. **Welcome** — brief intro
2. **Install Location** — default `C:\Program Files\HandVol\`, browse button
3. **Ready to Install** — shows chosen path and what will be installed
4. **Extraction** — progress bar as files extract, Task Scheduler entry created
5. **Finish**
   - Checkbox: "Run HandVol now?"
   - Checkbox: "Create desktop shortcut?"
   - "Finish" button

### Post-Install Actions
- If "Run now?" checked → launch `handvol.pyw` via `pythonw.exe`
- If "Create shortcut?" checked → add desktop shortcut pointing to `handvol.pyw`
- Task Scheduler automatically runs at next boot regardless

## Task Scheduler Setup

**Script:** `install_task.bat` (created in repo, called by NSIS during install)

```batch
REM Create Task Scheduler entry
schtasks /create /tn "HandVol" /tr "pythonw.exe C:\Program Files\HandVol\handvol.pyw" /sc onstart /rl highest /f /np
```

On uninstall, NSIS calls:
```batch
schtasks /delete /tn "HandVol" /f
```

Task will appear in Task Scheduler and can be toggled off/on by users without uninstalling the app.

---

## Uninstall Behavior

Triggered via Add/Remove Programs or direct `uninstall.exe`:

1. Stop any running `pythonw.exe` process for HandVol
2. Delete Task Scheduler entry `HandVol`
3. Delete directory `C:\Program Files\HandVol\` (all files)
4. Delete desktop shortcut (if it exists)
5. Remove registry entries under `HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Uninstall\HandVol`

No leftover files or registry keys.

## Build & Distribution

### Prerequisites
- NSIS installed (`makensis.exe` in PATH)
- Git LFS configured for the repo
- Python 3.11+ for build script

### Build Script
A `build_installer.py` (or `.bat`) will:
1. Clone/pull latest repo code
2. Download embeddable Python 3.11 if not cached
3. Pull MediaPipe model from Git LFS
4. Install pip packages into bundled Python
5. Run NSIS compiler to generate `.exe`
6. Output: `HandVol-Installer.exe` in `dist/`

### Distribution
- Upload `HandVol-Installer.exe` to a release page (GitHub Releases, etc.)
- Include checksum for verification
- Document system requirements (Windows 10/11, admin privileges)

## Compatibility

- **Windows:** Windows 10 (build 19041+) and Windows 11
- **Architecture:** x64 (embeddable Python is x64)
- **Admin:** Required for install and Task Scheduler access
- **Webcam:** Detected at app runtime (not at install time)
- **Audio:** pycaw handles device detection at runtime
